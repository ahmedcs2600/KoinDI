# Android KOIN DI Demo

Implementation of basic MVVM with Koin Dependency Injection using Kotlin DSL

## Libraries Used
* Koin (Dependency Injection)
* Navigation Component (for Fragment Navigation)
* Retrofit (For Consuming Rest API.
* Gson (JSON Parsing)
* Glide (Image Caching)
